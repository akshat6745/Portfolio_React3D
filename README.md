### Installation

1. Clone the repository
```bash

cd Portfolio_React3D
```

2. Install dependencies
```bash
npm install
```

3. Start the development server
```bash
npm start
# or
npm run dev
```

4. Open [http://localhost:5173](http://localhost:5173) to view it in the browser.
